# Find and Copy Files for a Specific User

## Scenario:-
On a Linux server, user data was mixed across directories.  
The task is to locate all files owned by a specific user and copy them to a new destination while preserving the original directory structure.

## Steps Taken:-
- Verified the existence of the target user.
- Identified all files (excluding directories) owned by the user.
- Copied the files to a new directory while preserving the directory hierarchy.

## Outcome:-
All files owned by the specified user were successfully copied to the target directory with the original directory structure intact.
