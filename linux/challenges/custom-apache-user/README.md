#create a custom apache-user

#Scenario:-
A linux system required a dedicated user account for an apache hosted application to improve application level isolation and security

#objectives:-
- create a custom linux user for an application
- assign a specific user id
- assign a non-default home directoy


#steps taken:-
- verifed the user exists or not
- created custom user id and home directoy under /var/www
- verifed ownership and configuration

#outcome:-
 successfully created a user account with dedicated is and homedirectory for better application level isolation and security
